Shader

Shader（着色器）是一种计算机程序，主要用于在图形渲染管线中对场景中的对象进行光照、材质等方面的渲染。Shader通常用于在GPU上实现高效的图形渲染。

Shader可以分为顶点着色器（Vertex Shader）、片元着色器（Fragment Shader）和几何着色器（Geometry Shader）等不同类型。顶点着色器用于对每个顶点进行处理，执行顶点变换、法线计算、切线计算等操作；片元着色器用于对每个像素进行处理，计算光照、材质、纹理等操作；几何着色器则用于对图元进行处理，可以对输入的图元进行修改和生成新的图元。

Shader的编写通常使用OpenGL Shader Language（GLSL）或High Level Shader Language（HLSL）等着色器语言。这些着色器语言提供了一些内置变量和函数，以便程序员能够轻松地访问图形渲染管线中的各个阶段。

在实际应用中，Shader可以用于实现各种图形效果，如反射、折射、阴影、光照、材质等。因此，Shader是图形渲染中不可或缺的一部分，对于实现高质量的图形渲染效果具有非常重要的作用。

模型空间、世界空间、观察空间和裁剪空间

模型空间、世界空间、观察空间和裁剪空间是计算机图形学中常见的坐标空间，它们用于描述三维物体的位置和方向，并且在渲染管线中扮演重要角色。

  • 模型空间（Model Space）：是指物体的本地坐标系，即在物体自身坐标系下描述的坐标。在模型空间中，物体的原点一般位于其几何中心。
  • 世界空间（World Space）：是指在全局坐标系下描述的坐标系，所有物体都共享同一个世界坐标系。在世界空间中，物体的位置是相对于全局原点的。
  • 观察空间（View Space/Camera Space）：是指相机拍摄时的坐标系，相机位于坐标系原点，观察方向为z轴负方向，x轴和y轴分别表示相机视野中的水平和垂直方向。
  • 裁剪空间（Clip Space）：是指相机空间中的坐标经过投影变换（投影矩阵）后得到的坐标系。在裁剪空间中，所有坐标都在范围[-1,1]之间。这个坐标系是为了方便进行透视变换，把物体从三维空间投影到二维屏幕上。

在渲染管线中，顶点着色器把模型空间中的顶点转换为裁剪空间中的顶点，然后光栅化阶段把裁剪空间中的几何图元转换为屏幕空间中的像素，最终显示在屏幕上。在这个过程中，我们需要对坐标进行多次变换和计算，以便在不同的坐标空间中进行正确的渲染和投影。

Mipmap（多级纹理贴图）是一种在计算机图形学中用于纹理映射的技术。它通过创建纹理的预缩小版本，以提高渲染效率和减少纹理映射的视觉伪像。

Mipmap使用了一系列不同分辨率的纹理图像，从原始纹理的全分辨率开始，逐渐缩小到最低分辨率。每个级别的纹理图像都是前一个级别的1/4大小。这些预缩小的图像被存储在一个纹理集合中，称为Mipmap链。在渲染时，根据物体与相机之间的距离，选择合适的Mipmap级别来进行纹理采样。

Mipmap的优点包括：

  1. 减少了纹理采样时的失真和锯齿现象，提高了纹理映射的质量。
  2. 在远距离观察时，可以使用较小的纹理级别，提高了渲染性能。
  3. 提供了平滑的过渡，避免了纹理跳变的不连续性。

Mipmap在许多计算机图形应用中被广泛使用，特别是在实时渲染、游戏开发和虚拟现实领域。它可以显著改善纹理映射的效果，同时减少对图形硬件的负载，提高了渲染的性能和视觉质量。

在Unity中，三维纹理（3D Textures）是一种特殊类型的纹理，与普通的二维纹理（2D Textures）不同，它具有额外的纹理维度。三维纹理可以用于在三维空间中进行体积渲染、体积阴影和其他体积相关的效果。

在Unity中创建和使用三维纹理的步骤如下：

  1. 创建三维纹理资源：在Unity编辑器中，你可以创建一个新的三维纹理资源。选择Assets菜单 -> Create -> 3D Texture来创建一个空的三维纹理资源。
  2. 设置三维纹理属性：在创建的三维纹理资源上，你可以设置其属性，包括纹理的尺寸、格式和过滤方式等。你可以根据需求设置不同的属性。
  3. 编辑三维纹理数据：打开三维纹理资源，你可以编辑其像素数据。Unity提供了一些API来读取和修改三维纹理的像素值。你可以通过编写自定义的脚本来生成、修改或加载三维纹理的像素数据。
  4. 应用三维纹理：一旦你编辑好了三维纹理的像素数据，你可以将其应用到你的材质或渲染器上。在材质中使用三维纹理时，你可以将其作为主纹理或作为材质的某个属性来控制体积效果。
  5. 渲染和预览：在场景中，你可以使用相机和照明来渲染和预览使用三维纹理的效果。根据你的需求和使用的渲染技术，你可以调整相机和材质的设置来实现所需的效果。

需要注意的是，三维纹理在内存和计算资源方面相对较大，因为它需要存储额外的维度信息。在使用三维纹理时，请考虑其对性能和资源消耗的影响，并根据需要进行适当的优化。

几何的CSG表示（Constructive Solid Geometry）是一种用于表示和操作几何形状的方法，它通过使用基本几何操作（如并集、交集和差集）来组合简单的几何体，生成更复杂的几何形状。

在CSG表示中，几何形状被表示为一系列基本的几何体（也称为原始体素或基元），这些基元可以是球体、方体、圆柱体等简单的几何体。通过对这些基元应用基本几何操作，可以生成复杂的几何形状。

常见的CSG基本操作包括：

  1. 并集（Union）：将两个或多个几何体组合成一个新的几何体，结果是将它们的几何形状合并在一起。
  2. 交集（Intersection）：将两个或多个几何体的交集作为新的几何体，结果是保留它们重叠的部分。
  3. 差集（Difference）：从一个几何体中减去另一个几何体，结果是保留第一个几何体中不与第二个几何体重叠的部分。

CSG表示可以使用树结构来组织几何体和操作的关系。常见的CSG树结构包括二叉树或多叉树，其中树的节点表示基本几何体或操作，而树的分支表示操作的层次结构和组合方式。

CSG表示的优点包括：

  • 可以使用简单的基本几何体和操作来描述复杂的几何形状。
  • 具有完全参数化的属性，可以轻松修改和调整形状。
  • 可以进行几何形状的布尔运算和组合操作，方便形状的编辑和构造。

然而，CSG表示也具有一些限制和挑战，例如处理边界和表面拓扑、求解复杂几何形状的性能和效率等方面的问题。

在计算机图形学和计算机辅助设计中，CSG表示被广泛应用于建模、渲染、虚拟现实和计算机辅助制造等领域。

分形表示

类似于递归

几何的分形表示是一种用于描述自相似或部分自相似几何形状的方法。分形是指具有类似结构的复杂对象，其部分可以看作是整体的缩小或放大。

在几何的分形表示中，常用的方法之一是使用迭代函数系统（Iterated Function System，IFS）。迭代函数系统是一组数学函数，通过对初始点进 行迭代变换，生成分形结构。每个函数定义了一个缩放、旋转和平移等变换操作，这些操作在每一次迭代中应用于初始点，形成新的点。

具体地，几何的分形表示可以通过以下步骤实现：

  1. 定义迭代函数系统：选择一组适当的数学函数，这些函数描述了几何形状的变换规则。每个函数通常由缩放因子、旋转角度和平移向量等参数来控制。
  2. 选择初始点：选择一个初始点作为起始点，该点将通过迭代函数系统进行变换。
  3. 迭代变换：将初始点应用于迭代函数系统中的函数，生成新的点。重复这个过程，将生成的点再次应用于迭代函数系统，不断迭代下去。
  4. 终止条件：可以定义一个终止条件，当满足特定条件时停止迭代。例如，可以设置最大迭代次数或当生成的点达到一定精度范围时停止迭代。

通过迭代函数系统，可以生成具有自相似性质的分形形状。每次迭代会将初始点分解为更小的子集，这些子集具有与整体形状类似的结构。

除了迭代函数系统，还有其他方法可以用于几何的分形表示，如分形细分曲线、分形树和分形噪声等。这些方法利用数学算法和随机性质来生成具有分形特征的几何形状。

几何的分形表示在计算机图形学、艺术设计和自然模拟等领域中得到广泛应用。它可以用于创建逼真的自然景观、生成艺术作品以及模拟自然现象等。

贝塞尔曲线

给定控制点，生成贝塞尔曲线

贝塞尔曲线（Bézier curve）是一种常用的数学曲线，用于描述平滑的曲线路径。贝塞尔曲线以法国数学家皮埃尔·贝塞尔（Pierre Bézier）的名字命名，他在20世纪60年代开发了这种曲线表示方法。

贝塞尔曲线通过控制点来定义曲线的形状。曲线的起始点称为起始控制点（P0），曲线的终点称为结束控制点（P3）。另外，还有两个可选的控制点（P1和P2）用于控制曲线的形状。

当 t 取 0 时，曲线位于起始控制点 P0 处；当 t 取 1 时，曲线位于结束控制点 P3 处。控制点 P1 和 P2 的值会影响曲线的形状。

贝塞尔曲线的特点包括：

  • 平滑性：贝塞尔曲线通常是平滑的，没有突变或锐角。
  • 可变形：通过调整控制点的位置，可以改变曲线的形状，使其适应各种需求。
  • 局部控制：控制点的位置只影响相应部分曲线的形状，使得局部修改不会影响整体曲线。

贝塞尔曲线广泛应用于计算机图形学、计算机辅助设计（CAD）、动画和游戏开发等领域。它们可以用于绘制曲线、路径动画、字体设计、形状变形等。

两段贝塞尔曲线的连续

起点和终点位置匹配：确保前一段曲线的终点与后一段曲线的起点位置相匹配，以消除连接处的间隙。

1.切线方向匹配：通过调整连接处的控制点位置，使得相邻曲线段的切线方向连续匹配。以下是具体的步骤：

a. 计算前一段曲线的终点处切线方向：T1 = 3(P1 - P0) b. 计算后一段曲线的起点处切线方向：T2 = 3(P3 - P2) c. 将这两个切线方向进行匹配，使得它们相等或成比例。可以通过调整控制点的位置来实现这一点。具体做法是将后一段曲线的起点处控制点 P1’ 设置为 P3 - T2/3，将前一段曲线的终点处控制点 P2’ 设置为 P0 + T1/3。

2.曲线的二阶导数匹配：除了匹配切线方向外，还可以考虑匹配曲线在连接处的二阶导数（曲率）。这可以通过调整连接处的控制点位置来实现。具体做法是计算前一段曲线的终点处的二阶导数 D1 = 6(P2 - 2P1 + P0) 和后一段曲线的起点处的二阶导数 D2 = 6(P3 - 2P2 + P1)，然后使得 D1 = D2。这样可以确保曲线在连接处的曲率连续。

3.三阶导数匹配：除了匹配切线方向和曲线的曲率外，可以考虑匹配曲线在连接处的三阶导数（挠率）。通过调整连接处的控制点位置，使得前一段曲线和后一段曲线在连接处的三阶导数相等，以实现更高阶的连续性。

4.N 阶导数匹配：类似地，可以考虑匹配曲线在连接处的更高阶导数。通过逐级匹配导数直到 N 阶，可以实现 N 阶连续性。这种方法可以提供更高级别的曲线控制，但也会增加计算和控制的复杂性。

5.控制点权重匹配：除了调整控制点的位置，还可以考虑调整控制点的权重。通过给连接处的控制点分配适当的权重，可以改变曲线在连接处的形状，以实现更复杂的连续性要求。

6.连续性插值算法：还可以使用连续性插值算法，如Hermite插值或Catmull-Rom插值，来实现更高级的连续性。这些算法可以在连接处平滑地插值曲线段，以实现更复杂的曲线形状。

样条

样条具有局部性，当改变一个点时，只会改变某个范围内的曲线而不会改变整条曲线

曲线的样条是一种平滑连续的曲线表示方法，通常由一组控制点和一组定义曲线形状的插值或逼近条件组成。样条曲线可以用于生成复杂的曲线形状，并提供良好的平滑性和控制性。

常见的曲线样条包括B样条（B-spline）和NURBS（非均匀有理B样条）。

  1. B样条（B-spline）：B样条曲线由一组控制点和节点向量定义。节点向量定义了参数空间中的节点，而控制点则决定了曲线的形状。B样条曲线通过将连续的Bézier曲线片段连接在一起而形成。B样条曲线具有局部控制性，即修改一个控制点只会影响局部区域的曲线形状。
  2. NURBS（非均匀有理B样条）：NURBS曲线是一种扩展的B样条曲线，它引入了有理权重。每个控制点都有一个相关联的有理权重，可以调整曲线在该点处的弯曲和拉伸。NURBS曲线允许更灵活的控制曲线形状，并可以表示圆弧、椭圆等更复杂的曲线形状。

曲线样条通常具有以下特点：

  • 平滑性：样条曲线在相邻控制点之间具有平滑过渡，不会出现明显的角点或断裂。
  • 局部控制性：修改一个控制点只会影响局部区域的曲线形状，使得对曲线的修改更加灵活。
  • 参数化：样条曲线通常是基于参数化的表示，参数值决定曲线上的点位置。
  • 可变次数：通过调整控制点的数量和次数，可以生成不同阶数的样条曲线，从而实现对曲线复杂度和平滑性的控制。

样条曲线在计算机图形学、计算机辅助设计（CAD）、动画制作等领域得到广泛应用，用于绘制和生成各种复杂的曲线形状。它们提供了强大的工具和灵活性，使得用户能够精确地控制和调整曲线的形状和属性。

贝塞尔曲面

贝塞尔曲面是由贝塞尔曲线组成的曲面。类似于贝塞尔曲线，贝塞尔曲面也是通过控制点来定义形状的。它是一种常用的曲面表示方法，在计算机图形学和计算机辅助设计中广泛应用。

贝塞尔曲面的定义基于二维或三维的控制点矩阵。对于二维贝塞尔曲面，通常使用二维的控制点矩阵来定义。每个控制点具有两个坐标（x，y），形成一个二维矩阵。对于三维贝塞尔曲面，使用三维的控制点矩阵来定义。每个控制点具有三个坐标（x，y，z），形成一个三维矩阵。

贝塞尔曲面的形状由控制点及其权重决定。权重用于调整每个控制点对曲面的影响程度。贝塞尔曲面的形状在控制点之间平滑过渡，并受到每个控制点的位置和权重的影响。

贝塞尔曲面的生成涉及到控制点的插值或逼近。常见的方法包括：

  1. 均匀贝塞尔曲面（Uniform Bezier Surface）：使用均匀参数化的贝塞尔曲线对每行或每列的控制点进行插值。这种方法简单快速，但可能会导致曲面形状的扭曲。
  2. B样条曲面（B-spline Surface）：将贝塞尔曲线的控制点扩展到二维或三维，并使用B样条的插值方式生成曲面。B样条曲面提供了更高级的控制和平滑性，具有局部调整能力。
  3. NURBS曲面（Non-Uniform Rational B-spline Surface）：类似于NURBS曲线，NURBS曲面在B样条曲面的基础上引入了有理权重。这允许更灵活的曲面形状调整，并能够表示圆弧、椭圆等更复杂的形状。

  • 局部控制性：修改一个或多个控制点只会影响局部区域的曲面形状。

Loop细分

Loop细分是一种常见的网格细分算法，用于将初始粗糙的三角网格转换为更细密的网格。该算法由Charles Loop于1987年提出，广泛应用于计算机图形学中的建模和渲染。

Loop细分算法的基本思想是通过迭代地细分三角形面片来生成更多的细分点，并根据一定的规则调整细分点的位置，从而得到更光滑和细致的曲面。下面是Loop细分算法的基本步骤：

  1. 初始化：将初始的三角网格作为输入，包括顶点坐标和三角形面片的连接关系。
  2. 细分边：对每条边进行细分，通过在边的中点处插入一个新的顶点来细分边。这样可以将每个三角形细分为四个子三角形。

  1. 更新顶点位置：根据细分边的操作，计算每个顶点的新位置。新位置是根据相邻顶点的位置和权重来计算的，以保持曲面的光滑性。

对于新的顶点：获取周围两个三角形，得到ABCD四个坐标，根据ABCD的坐标更新新顶点的信息

对于旧的顶点：计算周围节点的权重和自己的权重

  1. 更新连接关系：根据细分边的操作，更新连接关系，包括顶点之间的连接关系和三角形面片之间的连接关系。
  2. 重复细分：重复执行步骤2至步骤4，直到达到所需的细分级别或满足终止条件。

Loop细分算法的特点包括：

  • 局部控制性：细分操作仅影响每条边和相邻顶点的局部区域，而不会对整个网格进行全局修改。
  • 光滑性：通过调整新插入的顶点位置，Loop细分算法可以生成光滑且接近原始曲面的细分结果。
  • 细节控制：通过控制细分级别或权重，可以调整细分后网格的细节和光滑程度。
  • 高效性：Loop细分算法在计算和存储方面相对高效，可以生成相对细密的网格。

Loop细分算法在建模和渲染中被广泛应用，用于生成光滑的曲面模型，例如人脸、动物角色等。它在细分曲面表示、细分曲面绘制和细分曲面渲染等方面具有重要作用。

边坍缩

边塌缩（Edge Collapse）是网格简化算法中的一种操作，用于减少三维网格的复杂度。该操作通过合并两个相邻顶点之间的边来减少网格的顶点数量。边塌缩的目标是在保持尽可能接近原始模型形状的前提下，尽量减少顶点数量。

在边塌缩操作中，通常会使用二次误差度量（Quadric Error Metrics）来衡量合并后的简化结果与原始模型的误差程度。二次误差度量是一种用于评估网格简化效果的度量方法，它基于顶点位置的变化和拓扑关系的变化。

具体而言，二次误差度量通过构建一个二次误差矩阵来表示每个顶点的误差，该矩阵考虑了顶点位置的变化对误差的影响。在边塌缩操作中，合并两个顶点时，将两个顶点的二次误差矩阵相加，得到合并后的顶点的二次误差矩阵。

通过最小化二次误差度量，可以选择最优的边进行塌缩，以达到简化网格的目标。常见的算法如QEM（Quadric Error Metrics）和HEM（Half-Edge Collapse）使用二次误差度量来指导边塌缩操作。

总结起来，边塌缩是一种网格简化操作，而二次误差度量则是一种用于评估简化效果的度量方法，在边塌缩操作中起到指导和优化的作用。

ShadowMaping

产生阴影：摄像机能看到这个点，但是光源不能看到这个点

Shadow mapping（阴影映射）是一种在计算机图形学中用于实现实时阴影效果的技术。它是基于光的投射原理，通过将场景从光源的视角渲染到一个特殊的纹理中，然后在渲染场景时利用该纹理来判断每个像素是否处于阴影中。

下面是Shadow mapping的基本步骤：

  1. 创建Shadow Map：从光源的视角渲染场景，并将深度信息存储在一个称为Shadow Map的纹理中。Shadow Map通常使用一个深度缓冲区来记录每个像素与光源之间的距离。

  2. 渲染场景：从观察者的视角渲染场景，同时利用Shadow Map来确定每个像素是否处于阴影中。

    a. 对于每个像素，将其从观察者的视角转换到光源的裁剪空间。

    b. 在Shadow Map中，通过比较当前像素的深度与Shadow Map中对应位置的深度值来确定是否处于阴影中。如果当前像素的深度值大于Shadow Map中的深度值，则说明该像素处于阴影中。

    c. 根据像素是否处于阴影中，进行相应的渲染操作，例如应用阴影颜色或调整亮度。

Shadow mapping的主要优点是它能够在实时渲染中实现动态阴影效果。然而，它也存在一些问题，例如阴影失真、精度损失以及自阴影等。为了解决这些问题，还有一些相关的技术和改进算法，如PCF（Percentage-Closer Filtering）和Variance Shadow Mapping等。

总的来说，Shadow mapping是一种常用的实时阴影技术，通过渲染场景并利用Shadow Map来确定像素的阴影状态，从而实现动态阴影效果。
